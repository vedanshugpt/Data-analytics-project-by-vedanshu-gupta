# HR Data Analytics Dashboard | Power BI & Tableau

This project focuses on analyzing an HR dataset to derive meaningful workforce insights and support data-driven HR decision-making.

The same dataset has been used to build dashboards in both **Power BI** and **Tableau** in order to compare visualization capabilities, dashboard design approaches, and analytical workflows across different BI tools.

## Project Objective

The objective of this project is to help the HR department monitor key workforce metrics such as employee count, attrition trends, age demographics, job satisfaction, and department-wise employee behavior.

This dashboard helps in identifying:

* Employee attrition patterns
* Department-wise turnover
* Gender-based attrition trends
* Age group distribution
* Education field-wise attrition
* Job satisfaction levels

## Key KPIs

* Total Employee Count
* Attrition Count
* Attrition Rate
* Active Employees
* Average Age

## Dashboard Insights

* Attrition by Gender
* Department-wise Attrition
* Number of Employees by Age Group
* Job Satisfaction Rating
* Education Field-wise Attrition
* Attrition Rate by Gender for Different Age Groups

## Tools Used

* Power BI
* Tableau
* Excel (Data Source)

## Purpose

This project was created as a hands-on practice and capstone-style project to strengthen skills in:

* Data Analytics
* Dashboard Design
* Business Intelligence
* Data Storytelling
* HR Data Analysis

## Outcome

By building the same project in both Power BI and Tableau, this project provides a deeper understanding of how different BI tools can be used to solve the same business problem effectively.
